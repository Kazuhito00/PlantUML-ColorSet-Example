# Requirement (Unity)
* V1.2021.5 or later <br>※最新版で確認しましたが以前のバージョンでも動くと思います

# Usage
使用したいカラーセットを !include で読み込んでください。<br>
```
@startuml
!include colorset/colorset01.txt

@enduml
```
